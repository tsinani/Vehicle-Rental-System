[5/29/2026 1:19 PM] Tsi: Vehicle Rental System A Java console application that simulates a basic vehicle rental service. Built as a learning project to practice core Object-Oriented Programming principles in a real-world scenario. What It Does This system lets you manage a small fleet of vehicles right from your terminal. You can add vehicles to the system, browse what's available, rent one out, and return it when you're done — all through a simple text-based menu. Features

Add new vehicles to the fleet View all currently available vehicles Rent a vehicle by ID Return a previously rented vehicle Basic input validation and error handling

Concepts Practiced This project was built around core Java and OOP fundamentals:

Abstraction & Inheritance — A base Vehicle class extended by specific vehicle types Encapsulation — Private fields accessed through getters/setters Polymorphism & Method Overriding — Shared behaviors implemented differently per vehicle type Exception Handling — Graceful responses to invalid operations ArrayList — Dynamic storage for the vehicle fleet Packages — Organized project structure across multiple packages Access Modifiers — Controlled visibility across classes
